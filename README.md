# online_store
Интернет магазин

Функциональность:

- Категория товаров

    - Описание категории 
    
    - Товары
    
  
- Товары 

    - Название 
    
    - Изображение 
    
    - Описание 
    
    - Цена 
    
    - Корзина 
    
    - Количество товара
    
    
- Корзина 

    - Изображение 
    
    - Название товара
    
    - Количество товара 
    
    - Удаление товара из корзины
    
    - Цена за 1 еденицу товара
    
    - Общая цена за N едениц товаров
    
    - Итоговая цена за все товары
    
    Оформить заказ

- Отправка письма с номером заказа, после покупки товаров

    Реализовано с помощью celery + redis
